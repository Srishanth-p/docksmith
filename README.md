# Docksmith Project

## Description
This project implements a simplified container runtime system.

## My Contribution
- Layer creation using tar
- SHA256 hashing for layers
- Image manifest system
- Image storage and retrieval

## Folder Structure
- ~/.docksmith/layers → stores layer files
- ~/.docksmith/images → stores image metadata

## Commands Used
- create_layer()
- store_layer()
- create_manifest()
- add_layer()
- save_manifest()
